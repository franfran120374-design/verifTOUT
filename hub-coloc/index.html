📅 Créer une Tâche ou un Événement

📋 Mes Tâches - Liste simple

Consultez toutes vos tâches en un coup d'oeil :

  • 🖱️ Cliquer sur le titre - Voir les détails complets (date, heure, créateur, statut)
  • ✏️ Éditer - Modifier titre, date, heure ou récurrence
  • ✖️ Supprimer - Supprimer la tâche et toutes ses récurrences
  • ☐ Marquer comme fait - Cocher une tâche complétée
💡 Cliquez sur le titre "Nettoyer cuisine" pour voir tous les détails : qui l'a créée, si c'est fait, qui l'a marquée comme faite, etc.
📆 Calendrier visuel

Voyez toutes vos tâches et événements dans un vrai calendrier :

  • ◀▶ Naviguez mois par mois
  • Tâches en bleu, Événements en rouge
  • Les récurrences s'affichent automatiquement chaque jour concerné
  • 🖱️ Vous pouvez aussi cliquer sur les tâches/événements dans le calendrier
⚡ Créer une tâche récurrente

Exemple : "Nettoyer la cuisine tous les mardis à 19h"

1. Titre : "Nettoyer cuisine"
2. Date : Aujourd'hui
3. Heure : 19:00
4. Fréquence : "Chaque semaine" → "Mardi"
5. La tâche s'affichera automatiquement chaque mardi !
👥 Voir les détails d'un événement

Quand tu crées un événement (Présence), cliquer sur le titre te montre :

  • ✅ Qui a dit "Présent"
  • ❌ Qui a dit "Absent"
  • 📅 Date et heure exactes
  • 👤 Qui a créé l'événement

🛒 Créez une liste de courses partagée

Tous les colocs peuvent ajouter des articles :

Au-dessus du formulaire, trouvez les articles fréquents (Lait, Pain, Oeufs, etc.). Cliquez un bouton pour l'ajouter instantanément !

✅ Cochez les articles achetés

📊 Évolution annuelle

💧 Ajouter un relevé d'Index

📊 Visualisez vos consommations

Le graphique montre l'évolution annuelle de :

  • ⚡ Électricité (Heures Pleines & Heures Creuses)
  • 💧 Eau
  • 🔥 Gaz
Sélectionnez l'année avec le filtre en haut
🔢 Enregistrez un relevé d'index

Chaque mois (ou à chaque lecture du compteur) :

1. Type : Sélectionnez "Électricité", "Eau", ou "Gaz"
2. Index : Notez le chiffre du compteur
3. Date du relevé
4. (Optionnel) Joignez une photo du compteur
5. Cliquez "Enregistrer le relevé"
⚡ Pour les 2 tarifs électricité

Si vous avez Heures Pleines (HP) et Heures Creuses (HC) :

  • Enregistrez d'abord l'HP
  • Puis l'HC dans un 2e relevé
🤓 Analyser les tendances

Le graphique vous montre si vos consos augmentent ou baissent. Aidez-vous à réduire les dépenses d'énergie en colocation ! ♻️
