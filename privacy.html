Politique de Confidentialité

Dernière mise à jour : 20 mars 2026

L'application Hub Coloc 2026 est une plateforme de gestion collaborative pour les colocations. Nous accordons une importance capitale à la protection de vos données privées.

1. Données collectées

Nous collectons les informations suivantes uniquement dans le but d'assurer le fonctionnement de la colocation :

  • Identité : Votre prénom (pour vous identifier dans le chat et les finances).
  • Communications : Les messages envoyés sur le chat interne.
  • Contenus : Les photos de tickets de caisse ou documents joints aux factures.
  • Données techniques : Le nom de votre colocation (token) pour regrouper les membres.

2. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Synchroniser les informations entre les membres d'une même colocation.
  • Calculer les dettes et remboursements de manière équitable.
  • Signaler des bugs techniques via notre système de support.

3. Stockage et Sécurité

Vos données sont stockées de manière sécurisée sur les serveurs de Google Firebase (Région Europe). Nous ne vendons, n'échangeons ni ne transférons vos données personnelles à des tiers à des fins commerciales.

4. Vos droits

Conformément au RGPD, vous disposez d'un droit d'accès, de modification et de suppression de vos données. Vous pouvez supprimer vos messages ou factures directement dans l'application, ou nous contacter pour une suppression totale du compte de colocation.

5. Contact

Pour toute question concernant cette politique ou pour signaler un problème, vous pouvez utiliser l'onglet "Support" de l'application ou nous contacter par email.
